# Sales Performance Dashboard

**Tools Used:** Power BI, Excel  
**Date:** December 2024

## Project Overview
Interactive sales dashboard analyzing product performance, category distribution, and sales trends across January-March 2024.

## Key Features
- Dynamic filtering with category and date slicers
- Monthly sales trend analysis
- Product performance comparison
- Revenue distribution by category

## Key Insights
- **Total Revenue:** 138K
- **Top Product:** Laptop (100K revenue, 72.5% of total)
- **Category Split:** Electronics 85.45%, Furniture 14.55%
- **Trend:** Declining sales from January (59K) to March (25K)
